Krita ComfyUI

What this plugin does:

  • Connects Krita to a remote ComfyUI server.
  • Lets you pick a workflow, enter a prompt and generate images.
  • Shows thumbnails of the results and lets you drop an image into the current document.

Usage

Steps to use the plugin:

  1. Open the “Krita ComfyUi” dock widget from Settings ▸ Dockers ▸ Krita ComfyUi.
  2. Enter or paste your ComfyUI server URL (e.g. http://localhost:8188/) in the “General” tab.
  3. Switch to the “Workflow” tab, pick a workflow and configure its inputs.
  4. Write a prompt in the text box below.
  5. Click Generar. A progress bar will show the status.
  6. When thumbnails appear, select one and click Apply to create a paint layer with that image.
