Krita ComfyUI

What this plugin does:

  • Connects Krita to a running ComfyUI server.
  • Lets you pick a workflow, enter a prompt and generate images directly from the Krita UI.
  • Shows thumbnails of the results and lets you drop an image into the current document.

Usage

Steps to use the plugin:

  1. Open the “Krita ComfyUi” dock widget from Settings ▸ Dockers ▸ Krita ComfyUi.
  2. Click the settings button at the top of the dock to open the configuration window.
  3. In the **General** tab, paste your ComfyUI server URL (e.g. http://localhost:8188/). Tip: If you run a local instance, start it before launching Krita so the plug‑in can auto‑connect.
  4. In the **Workflow** tab, choose a workflow and configure its inputs.
  5. Enter a prompt in the text box.
  6. Click Generate. A progress bar will show the status.
  7. When thumbnails appear, select one and click Apply to create a paint layer with that image.
